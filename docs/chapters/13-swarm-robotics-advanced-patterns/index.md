# Swarm Robotics and Advanced Engineering Patterns

## Summary

This capstone chapter pairs multi-robot coordination with advanced software engineering
patterns. Students program leader/follower BLE robots, implement collective obstacle
avoidance and convoy following, and explore emergent behaviors and distributed systems
thinking. The chapter also covers the advanced patterns that power professional robot
software: project planning and team collaboration, state machines for structured
behavior control, multithreading and asynchronous programming, PID control for
precision motor feedback, encoder-based odometry, and data logging for analysis.
