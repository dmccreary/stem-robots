Raspberry Pi Pico-based\n- **Programming Language**: MicroPython\n- **Key Sensors**: VL53L0X Time-of-Flight ($3), HC-SR04 Ultrasonic, IR sensors, bump switches\n- **Actuators**: DC motors, servo motors, NeoPixel LEDs, onboard buzzer\n- **Total Cost**: Under $19 per complete robot kit\n\n### Why This Hardware?\n\n- **No Soldering Required** -
